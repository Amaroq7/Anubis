◆ VERSION

constexpr std::uint32_t Metamod::IMetamod::VERSION = (MAJOR_VERSION << 16 | MINOR_VERSION)
staticconstexpr

Metamod API version.

Major version is present in the 16 most significant bits. Minor version is present in the 16 least significant bits.

Examples

To extract the major version of Metamod API the following formula can be used

std::uint16_t majorVer = (VERSION >> 16) & 0xFFFF;
